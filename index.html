もうEnterで送信しない。思考を止めない、Mac用テキスト入力パッド

CalmTypeは日本語入力時の誤送信を防ぎ、安心してテキストを作成できるユーティリティです。

Mac App Store でダウンロード

Enterキーでの「ヒヤリ」、もう終わりにしませんか？

フォーム入力中、SNSやチャットで返信中に日本語確定のEnterで意図せず送信…その瞬間のヒヤリとした経験はありませんか？ 改行したいだけなのに、送信されてしまったメッセージに冷や汗をかいたことは多くのMacユーザーに共通するストレスです。

CalmTypeで、穏やかな入力を。

グローバルショートカットで専用ウィンドウを呼び出し、その中で安全にテキストを作成。確定キーを別に設けることで、誤送信の不安から解放されます。

CalmTypeの主な機能

⌨️

どこからでも起動

Ctrl + Option + T (カスタマイズ可) で瞬時にウィンドウ表示/非表示。

📝

集中できる入力エリア

フローティングするミニマルなウィンドウ。自動フォーカスで即入力可能。

📋

確実なコピー＆クローズ

Cmd + Enterで内容をクリップボードへコピーし、そのままウィンドウを閉じます。

⚙️

自由なショートカット

設定画面 (Cmd + ,) で呼び出しキーを簡単に変更可能。

3ステップで簡単利用

ショートカットで呼び出し

任意の場所でCtrl + Option + Tを押してCalmTypeを開きます。

テキストを入力

ミニマルなウィンドウで集中して文章を作成します。

Cmd+Enterでコピー＆終了

Cmd + Enterを押すとクリップボードにコピーされ、ウィンドウが閉じます。
